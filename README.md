# WebPortfolio
Investigar el trabajo soñado del Ingeniero(a) en Sistemas a partir del análisis de ofertas laborales reales y, con base en dicha investigación, diseñar y desarrollar un sitio web tipo portafolio/CV profesional o página de servicios, utilizando HTML5 y CSS3, aplicando principios de diseño web, buenas prácticas de desarrollo y diseño responsivo.

# Empresa 1
## Nombre del puesto:
- Desarrollador Java Full Stack

## Tecnologías requeridas
- Backend: Java, Spring Boot, Spring Cloud.
- Frontend: React, Angular o frameworks JavaScript similares.
- Bases de Datos: Relacionales (SQL) o NoSQL.
- Fundamentos Web: HTTP, DOM, SSL, Servidores Web.
- Arquitectura: API, Enrutamiento, Almacenamiento de datos, Caché.

## Habilidades técnicas (Hard skills)
- Desarrollo de Software: Escritura de código mantenible, testable y de alto - rendimiento.
- Pruebas y Calidad: Realización de pruebas exhaustivas y revisión de código (code review).
- Ingeniería Web: Profundo conocimiento en patrones de diseño, optimización y seguridad.
- Investigación: Capacidad para investigar nuevas tecnologías y herramientas.
Experiencia: 1 a 3 años en aplicaciones web modernas en entornos de producción.

## Habilidades blandas (Soft skills)
- Comunicación: Capacidad de comunicarse de forma clara y eficaz.
- Trabajo en Equipo: Colaboración estrecha con equipos multidisciplinarios.
- Proactividad: Iniciativa y deseo constante de ofrecer un trabajo de alta calidad.
- Analítica: Pensamiento lógico y resolución de problemas complejos.
- Mejora Continua: Afición por optimizar flujos de trabajo y procesos de desarrollo.

## Responsabilidades principales
- Desarrollo y Entrega: Colaborar en la creación de aplicaciones para consumidores globales y empresas.
- Calidad del Código: Escribir software de alto desempeño y participar activamente en revisiones técnicas.
- Colaboración: Trabajar de la mano con el equipo de desarrollo para garantizar entregas exitosas.
